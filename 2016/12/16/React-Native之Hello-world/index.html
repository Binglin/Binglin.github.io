#开发环境
假定你已安装atom, 并安装atom的Nucide插件
atom使用指南

调用控制面板Cmd+Shift-P

#所需环境

  • node brew install node

  • watchman brew install watchman

  • React Native CLI npm install -g react-native-cli

  • XCode

测试React Native是否安装

#运行
运行React Native服务器: 选择Nucide React Native: Start Packager

服务器默认端口为8081

运行iOS

也可直接在XCode 或 Nucile中运行

调试

Nucide 支持iOS的 React Native的调试，但是调试器不会显示异常
